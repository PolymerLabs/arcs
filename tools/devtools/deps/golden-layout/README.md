## Installation

Add `golden-layout` to your bower.json, or [download](https://golden-layout.com/download/) the source.

## Features

* Native popup windows
* Completely themeable
* Comprehensive API
* Powerful persistence
* Works in IE8+, Firefox, Chrome
* Reponsive design
